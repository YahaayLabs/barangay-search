# Changelog

## [0.1.0] - 2026-07-10

### Added
- Initial `<barangay-search>` custom element (Shadow DOM).
- Debounced search via `gis.ph-sdk` / `api.gis.ph`.
- Events: `select`, `error`, `clear`.
- Attributes: `api-key`, `access-token`, `placeholder`, `disabled`, `clearable`, `debounce-ms`, `min-query-length`.
- Keyboard navigation (ArrowUp/Down, Enter, Escape).
- Playground checkout demo.
