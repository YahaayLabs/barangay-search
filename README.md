# barangay-search

Web component for Philippine **barangay autocomplete** — powered by the [GIS.ph](https://gis.ph) API.

```html
<script type="module" src="https://cdn.jsdelivr.net/npm/barangay-search/…"></script>
<barangay-search api-key="gis_sk_…"></barangay-search>
```

> Scaffold / early public repo. Implementation coming soon.
